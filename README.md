# About Me App

A simple Flutter personal introduction application.

## Features

- Personal profile section
- Avatar display
- About Me card
- Skills horizontal list
- Contact information
- Custom fonts and UI styling

## Built With

- Flutter
- Dart
- Material Design

## Project Structure

```text
lib/
 └── main.dart
assets/
 └── profile.png
```

## Getting Started

### Requirements

- Flutter SDK
- Dart SDK
- Android Studio / VS Code

### Run the Project

```bash
flutter pub get
flutter run
```

## Assets

Make sure your image asset is configured in `pubspec.yaml`.

Example:

```yaml
flutter:
  assets:
    - assets/profile.png
```

## Fonts

This project uses:

- Playfair Display
- Roboto

Remember to configure them in `pubspec.yaml`.
